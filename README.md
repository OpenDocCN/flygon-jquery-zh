## 下载

### Docker

```
docker pull apachecn0/flygon-jquery-zh
docker run -tid -p <port>:80 apachecn0/flygon-jquery-zh
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g flygon-jquery-zh
flygon-jquery-zh <port>
# 访问 http://localhost:{port} 查看文档
```
